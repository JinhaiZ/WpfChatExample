# Travaux Pratique pour .NET plate-form : programmation réseau

une application classique de chat-room en utilisant le frame-work .NET Remoting

![image](https://user-images.githubusercontent.com/15102793/33517474-1f08f9ac-d785-11e7-84db-cf1b136a2fba.png)


Voici les programmes pour le 3eme travail pratique dans la cadre de cours 
.NET plate-form. Dans ce projet, il y un programme client et un programme
serveur, le client a une interface utilisatuer, cependant  le serveur est
 en mode Console

## Comment utiliser ?

Importez le projet 

1. File -> Open -> Project/Solution -> choisissiez `Client.sln`
2. Lance le programme par cliquer le bouton `Start`

Ou allez dans le répertoire `WpfChatExample\Client\Client\bin\Debug` et 
`WpfChatExample\Client\Server\bin\Debug`, puis lancez `Server.exe` et 
`Client.exe` respectivement
